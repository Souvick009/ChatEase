# ChatEase - A Secure & User-Friendly Chat App  

ChatEase is a **real-time messaging app** built with **Native Android (Kotlin)** and **Firebase**, using **legacy Android components**.  

## ✨ Features  
- **Secure Authentication** (Firebase Authentication)  
- **Real-time Chat** (Firebase Realtime Database)  
- **Friend System** (Users must add each other before creating or adding them into the group)  
- **Recent Chats** (Stored locally using Firebase for quick retrieval)
- **Smart Notifications**:
  - Uses **Firebase Cloud Messaging (FCM)** for offline users  
  - Local notifications for online users to avoid duplicate alerts  
- **Optimized Data Fetching** (Delays search feature to prevent spam API requests)  
- **User Profiles** (Username-based search,shows profile pictures, display names, last seen status, bio)  
- **Read Receipts & Last Seen Status**  

## 🏗 Tech Stack  
- **Language**: XML & Kotlin 
- **UI**: Legacy Android components (Activities, Fragments, ListView, SharedPreferences, etc)  
- **Backend**: Firebase (Auth, Realtime Database, FCM, Storage)  
- **Networking**: Firebase SDK
